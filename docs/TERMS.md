# Velance Terms

Last updated: 2026-05-05

Velance is productivity analytics and focus coaching software. It is not medical, legal, financial, or clinical advice.

## Use

You are responsible for choosing whether to enable tracking, browser capture, AI insights, backup export, and cloud sync. Use Velance only on devices and accounts where you have permission to collect productivity data.

## AI Output

AI-generated insights can be incomplete or wrong. Treat them as coaching suggestions, not guarantees.

## Local And Cloud Data

Velance stores workspace data locally first. Optional cloud sync depends on Supabase availability and your signed-in account.

## Browser Extension

The browser extension is optional. Enabling browser access lets Velance collect browser context according to the selected capture mode.

## Public Release Status

Before public launch, every item in `docs/RELEASE_CHECKLIST.md` must pass.
